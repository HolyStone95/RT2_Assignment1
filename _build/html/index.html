execute_cb(goal)[source]
This calback takes as argument the goal variable whose value is

provided from the action client in the state_machine.cpp. The feedback of the action message is constantly updated as the current pose of the robot. The desired_position is initialised as the goal of the action. To conclude with, the change_state function (which is responsible for the _state assignment) is given with argument zero so that the robot can start fixing its own yaw before proceeding by reaching the goal
